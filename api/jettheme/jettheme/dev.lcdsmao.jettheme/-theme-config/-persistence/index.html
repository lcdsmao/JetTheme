@Immutable()
data class Persistence<T : ThemeSpec>(themePack: ThemePack<T>, persistenceKey: String?, darkModeThemeId: String) : ThemeConfig<T>
Content copied to clipboard

Configure the ThemeController to save theme preference (theme id) into local storage. If there is no preference, then ThemeController will choose the proper theme based on isSystemInDarkTheme.

Parameters

persistenceKey

the local storage key to store theme preference. If the value is null then a default key will be used.

darkModeThemeId

use this theme if there is no preference and isSystemInDarkTheme is true.
